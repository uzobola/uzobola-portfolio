# Security Policy

This repository is a portfolio project. Please do not submit sensitive data, credentials, or customer information.

## Reporting a vulnerability

Email: uzo@uzobolarinwa.com

## Secrets

This project is designed to run without committed secrets. Use environment variables, GitHub OIDC, or local AWS profiles where applicable.

## Supported use

This project demonstrates security patterns and should be reviewed and adapted before any production use. It is intended for educational and portfolio demonstration purposes.
